Linux Commands - File Operations and Compression

find Command

The find command is used to search for files and directories within a file system based on conditions like name, type, size, and more.

tar Command

The tar command is used to create, extract, and manipulate archive files. It can bundle multiple files and directories into a single file.

gzip Command

The gzip command is used to compress files. It is commonly used for reducing the size of files for storage or transfer.

gunzip Command

The gunzip command is used to decompress files that were compressed using the gzip command.

zip Command

The zip command is used to create compressed archives in the ZIP file format.

chmod Command

The chmod command is used to change the file permissions for files and directories.

chown Command

The chown command is used to change the ownership of a file or directory.

chgrp Command

The chgrp command is used to change the group ownership of a file or directory.

ps Command

The ps command is used to display information about the currently running processes on a system.
