Linux Command: gzip

Description:

The gzip command is used to compress files to reduce their size, making them easier to store and transfer.

Purpose & Usage:

Purpose: Compress files into .gz format for storage or transmission.

Usage: Commonly used in combination with tar for compressed archives.

Examples:

Compress a file:

gzip myfile.txt

This command compresses the myfile.txt file and creates a compressed file myfile.txt.gz.

Compress multiple files:

gzip file1.txt file2.txt

This command compresses both file1.txt and file2.txt, creating file1.txt.gz and file2.txt.gz respectively.

Explanation:

Common Use Case: Reducing the size of log files or backups.

Importance: Significantly reduces storage space for large files.
