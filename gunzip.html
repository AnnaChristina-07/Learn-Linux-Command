Linux Command: gunzip

Description:

The gunzip command is used to decompress files that were compressed using gzip. It restores the original file format.

Purpose & Usage:

Purpose: To decompress .gz files.

Usage: Used to retrieve the original file from a .gz compressed version.

Examples:

Decompress a .gz file:

gunzip myfile.txt.gz

This command decompresses the myfile.txt.gz file and restores it to its original myfile.txt format.

Explanation:

Common Use Case: Restoring compressed files to their original form.

Importance: Essential for working with compressed archives and files.
