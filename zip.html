Linux Command: zip

Description:

The zip command is used to compress files and directories into .zip archive files. It's one of the most widely used compression formats.

Purpose & Usage:

Purpose: To create compressed .zip files.

Usage: Compressing files for distribution or backup.

Examples:

Create a .zip archive from multiple files:

zip archive.zip file1.txt file2.txt

This command creates a archive.zip archive containing file1.txt and file2.txt.

Zip a directory recursively:

zip -r archive.zip /path/to/dir

This command creates a archive.zip archive from the specified directory and its contents.

Explanation:

Common Use Case: Creating compressed archives for storage or sharing.

Importance: .zip is compatible with many operating systems and is widely supported across different platforms.
